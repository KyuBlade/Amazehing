Item Template


Components :


IdentityComponent

NameComponent : name of the item

LevelComponent : level of the item

StackableComponent : if stackable or not

ItemTypeComponent : HEAD, NECKLACE, ARMOR, RING, WEAPON, LEGS, FEET, RESOURCE

TextureRegionComponent (Need to be changed) : represent the icon in the inventory


Weapons :


TwoHandedComponent : does weapon use both hand?

RangeComponent : range of attack

DamageComponent : damages that the weapon inflict


Consumables :


DestroyOnUseComponent : destroyed at first use?

CoolDownComponent : cooldown between uses

CastTimeComponent : time to use the item
