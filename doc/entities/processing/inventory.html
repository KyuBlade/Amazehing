Inventory Operations


Add items :


Components :


InventoryOperationComponent : ADD

IdentityComponent : item id to add

QuantityComponent : quantity to add

CallbackComponent (optional) : Called when finished


Remove items :


Components :


InventoryOperationComponent : REMOVE

IdentityComponent : item id to remove

QuantityComponent : quantity to remove

CallbackComponent (optional) : Called when finished


Sort items :


Components :


InventoryOperationComponent : SORT


System : InventorySystem
