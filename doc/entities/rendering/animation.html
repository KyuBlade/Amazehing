Animation



Components :



AnimationComponent : used to process the animation

TextureComponent : store the current animation frame to render

PositionComponent : position in the world

SizeComponent : scale the animation

RotationComponent : rotate the animation

FlipComponent : flip verticaly/horizontaly

ZindexComponent : render order

BlendComponent : if true active the opengl blending mode (performances)



Systems :



AnimationSystem : calculate the current animation frame from the state time and set the TextureComponent.

RenderSystem : render the current animation frame contained in the TextureComponent.
