Shape


Components :

ShapeRenderComponent : type of shape (rectangle, circle, polygon)

ColorComponent : color of the shape

PositionComponent : position of the shape in the world

ZIndexComponent : rendering order

BlendComponent : true to activate the opengl blending mode (performances)


Rectangle :

SizeComponent : size of the rectangle


Circle :

RadiusComponent : radius of the circle


Line/Polyline :

VerticeComponent : vertices forming the shape


System :

RenderSystem
